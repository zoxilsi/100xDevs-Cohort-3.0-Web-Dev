# 100xDevs-Cohort-3.0-Web-Dev


100xDevs Cohort 3.0 is an intensive, hands-on web development and DevOps program led by Harkirat Singh. Launched on August 2, 2024, the cohort is designed to take learners from foundational web development skills to advanced backend and DevOps practices, with a strong emphasis on real-world projects and industry-relevant tools.
